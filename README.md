# Wordle In Hebrew

"Wordle In Hebrew" is an engaging word-guessing game where players aim to decipher a 5-letter Hebrew word within a limited number of attempts.

## How to Play

1.  **Guess the Word:** Enter a 5-letter Hebrew word as your guess.
2.  **Receive Feedback:** After each guess, the tiles will change color to provide feedback:
    *   <span style="color: green; font-weight: bold;">Green</span>: The letter is in the correct place.
    *   <span style="color: yellow; font-weight: bold;">Yellow</span>: The letter is in the word but in the wrong place.
    *   <span style="color: gray; font-weight: bold;">Gray</span>: The letter is not in the word at all.
3.  **Earn Diamonds:** At the end of each round, you'll earn diamonds based on your remaining attempts.
4.  **Use Hints:** Spend 3 diamonds to get a hint.

## Setup and Installation

This is a web-based game. To play:

1.  Clone this repository to your local machine.
2.  Open `index.html` in your web browser.

Enjoy the game!
